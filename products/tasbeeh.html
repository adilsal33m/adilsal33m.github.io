A modern and ease to use Tasbeeh app for making daily zikr.

App Features

Predefined Daily Zikrs

Tasbeeh app comes with predefined common zikrs that you can recite daily.

Immersive View

Tasbeeh app also has a full screen immersive mode so that you can do zikr without any distraction

Zikr Details

While selecting a zikr, long press any zikr to view the translation, transliteration, virtue and accurate reference for the zikr.

Set Target

Set different target for different zikrs. The counts are maintained even when you close the app.

Beautiful Themes

Tasbeeh app's interface comes with multiple beautiful themes to suit your needs.
